3. В программе "консольный чат" пользователь может осуществить:
   -регистрацию - ввести логин, пароль, имя
   -вход по логину и паролю
   -отправить сообщение конкретному пользователю
   -отправить сообщение в общий чат
   -выйти из учетной записи(разлогиниться)
